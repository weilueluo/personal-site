# Personal Website

### Screenshot
![website screenshot](https://user-images.githubusercontent.com/39546701/209574676-94799146-f218-409e-9c15-eae51916fc92.png)

### Run Locally
```
git clone https://github.com/Redcxx/personal-website.git
cd personal-website
npm install
npm run build
npm run start
```
